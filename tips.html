Tips and Tricks

Getting Started 
 To get started, insert nodes on either the 'Exists' or 'Forall' side by clicking anywhere in the canvas but not in the top menu bar or very close to the vertical dividing line. A node will be created at the click point.
Selecting Nodes and Edges 
 To select a node or group of nodes, click (or cmd+click/shift+click) on the already-created node(s).
 To select an undirected edge, click anywhere within the edge.
 To select a directed edge, click anywhere within the arrowhead.
 To select a group of nodes and/or edges, either select the nodes and edges individually, or click and drag your mouse to see a selection rectangle. When you release your mouse, everything within the selection rectangle will be selected.
Creating Edges 
 To create an undirected edge between nodes, select the nodes and press the 'e' key (for Edge).
 To create a directed edge between nodes, select the nodes and press the 'd' key (for Directed edge). The edge will be directed from the first node selected to the second node selected.
 NOTE: all edges in a graph must be of the same type, in other words, either all edges must be directed or all edges must be undirected.
 To switch the direction of one or more directed edges, selected the edge(s) and press the 'r' key (for Reverse).
 To add a directed edge(s) in the opposite direction from a pre-existing edge(s) (thereby, effectively, making the edge undirected), select the edge(s) and press the 'R' key (for Reverse++).
 To create a self-loop, right click on a node and select Create a Self-Loop. You may then choose which side to draw the self-loop on. Once a self-loop has been created, you can move it to another side by using the same context menu. Delete a self-loop by clicking on the edge and deleitng it just like any other edge.
Deleting Nodes and Edges 
 To delete a node or group of nodes, select them and then either select Delete from the Edit menu or press the Delete key.
 To delete an edge or group of edges, select them and then either select Delete from the Edit menu or press the Delete key.
 To delete a collection of nodes and edges at the same time, simply select such each member of the collection, and either pressthe Delete key or select Delete from the Edit menu.
Cut, Copy and Paste 
 Any group of selected objects may be cut or copied by selecting the appropriate item from the Edit menu. (Keyboard shortcuts are not available in all browsers.)
 If you cut or copy objects on both sides, the target of a paste action will be both sides. If you cut or copy from just one side, you must indicate the target side to paste to. If you use a keyboard shortcut to paste (again, not available on all browsers), the target will be whichever side your mouse is on. If you choose to paste from the Edit menu, you must first select either the 'Exists Side' or 'Forall Side' buttons to indicate your preferred target.
Grouping and Moving 
 Once you have selected a set of nodes (possibly including some edges), you can group the nodes into one unit by selecting the Group item from the Edit menu (or by pressing the 'g' key). A grey colored rectangle with "handles" will then be drawn, indicating which nodes have been grouped together.
 A set of grouped nodes can be moved as a unit by grabbing onto the grouping rectangle and moving the nodes to a locaiton of your choosing on the canvas. Nodes cannot be moved from one side of the board to the other.
Iconized Groups 
 Once you have grouped a set of nodes and edges together, you can turn them into what we call an Iconized Group. Iconized groups must be completely disconnected from any other nodes on the canvas. This is useful if you want to have several copies of the same group of nodes and edges and display things more compactly and mnemonically. An icon will replace the set of nodes in the graphical display. Iconized groups can be copied and pasted anywhere on the canvas. Iconized groups can be created in different shapes, sizes and colors. They can also be given labels of up to 3 chracters.
 Iconoized groups can be connected to other nodes or iconized groups by clicking and typing 'e' (if edges are undirected) or 'd' (if edges are directed). This will form what we call an Ultra Edge between iconized group and any other node or iconized group that is also selected. This means that every node in the iconized group will be connected via an edge to all the other selected objects. Ultra edges are displayed with bold dashed lines.
 Iconized groups can be included within other iconized groups.
 Once an iconized group has been created it can be further edited right clicking and selecting Change Icon/Label, if just this is required, or by selecting Edit Group Contents, if that is required. The latter option is a good way to recall what the contents of an iconized group are if you have forgotten.
 An iconized group can be expanded to its original contents by right clicking and selecting Explode Iconized Group.
Ordering Nodes 
 To indicate that nodes should be ordered, right click on an empty space in the canvas and select Order Nodes. By default, nodes are not ordered.
 There is also a context menu option to Display Node Ordering. By default, nodes are ordered in the order in which they were created.
 Once orderings are displayed, they can be edited directly. There is also an option to order the set of selected nodes in left to right order. Node orderings are with respect to the other nodes on the same side.
Assigning Constants 
 To assign a constant to a node, right click on the node, select Assign a Constant. and input the desired constant.
 To delete a constant that has been assigned to a node, right click on the node, and select Remove Constant.
Inserting a Grid of Nodes 
 You may insert a rectangular array of nodes by right clicking at the location where you would like the nodes added, and then selecting Insert Grid of Nodes.... You may connect nodes to their L1 (vertical and horizontal) neighbors, or not, as desired.
Inserting a Cycle of Nodes 
 You may insert a cycle of nodes by right clicking at the location where you would like the cycle added, and then selecting Insert Cycle of Nodes.... You are able to specify the number of nodes in the cycle as well as the height and width of the cycle in pixels. You can connect neighboring node by directed or undirected edges.
Saving to/Retreiving from a File 
 You can save the current state of you work, or pickup where you left off in a prior session by selecting, respectively Save... or Open... from the File menu.
Starting a Game and Game Play 
 To start a game, select Start a Game from the Game menu. You must have nodes on both sides of the board to start a game. You can request that the computer play the role of Spoiler, Duplicator, both, or neither.
 When it is your turn, click on the node you wish to play on to register your move. If you wish to play within an iconized group, click on the iconized group and the iconized group will be expanded in a separate window. Click on the node within the separate window and the click on the Confirm button.
 Move numbers are indicated with different colored boxes atop the nodes or iconized groups.
 If you have requested that the computer play for either the Spoiler, Duplicator or both, various aspects of the computer's analysis will be dislayed in the lower left corner of the canvas, including when the computer has determined that one of the player's has a forced win.
 You can undo or redo a move by clicking on the associated undo and redo buttons beside the Rounds Thus Far display at the top of the canvas.
 You can get a view of the game, including a one-level explosion of any iconized groups by typing Command + 'e'.
 You can get a view of the game, including a complete (all-level) explosion of any iconized groups by typing Command + 'E'.
